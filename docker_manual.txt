============================================================
PJ_Elucidator: Docker Web 開発運用マニュアル
============================================================

【基本方針】
・編集：Windows 側の VS Code で行う（補完が効いて楽）
・同期：保存した瞬間にコンテナ内も更新される
・確認：ブラウザ（Chrome等）で localhost:3000 を開いて確認

------------------------------------------------------------
1. 開発の始め方（Webサーバー起動）
------------------------------------------------------------
プロジェクトのルート（docker-compose.yml がある場所）で実行：

$ docker compose up -d

※ 起動後、ブラウザで http://localhost:3000 を開くとページが表示されます。

------------------------------------------------------------
2. 修正とテスト（Web 開発ループ）
------------------------------------------------------------
1. Windows 側の VS Code で index.html や app.js を編集して「保存」
2. ブラウザを「リロード（F5）」して変更を確認
3. コンテナの中に入る必要はほとんどありません

------------------------------------------------------------
3. 新しいライブラリ（Three.js 等）を増やしたい時
------------------------------------------------------------
コンテナの中（Node.js 環境）にインストールを命令します：

$ docker exec -it elucidator-web npm install <パッケージ名>

※ これにより package.json が更新され、ボリューム同期で Windows 側にも反映されます。

------------------------------------------------------------
4. 開発の終わり方（電源 OFF）
------------------------------------------------------------
作業を中断する時：
$ docker compose stop

環境をリセットしたい、または設定を変えた時：
$ docker compose down

------------------------------------------------------------
【おまけ】よく使う確認コマンド
------------------------------------------------------------
・コンテナの稼働状況を確認：
$ docker compose ps

・サーバー側（Node.js）のエラーログを確認：
$ docker compose logs -f
